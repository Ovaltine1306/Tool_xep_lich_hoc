Quy định về quá trình làm dự án:
1. Tuyệt đối không push trực tiếp code vào nhánh main. Nhánh main sẽ chỉ chứa những code đã chạy được và không xảy ra lỗi.
2. Mỗi người tạo 1 nhánh riêng của mình để code và test chức năng của mình. Chạy ổn thì gửi pull request để anh em đọc qua và duyệt.
3. Trong thư mục chung thì /src dùng để chứa mã nguồn, /include dùng để chứa file header, /data dùng để chứa dữ liệu, /external dùng để chứa thư viện ngoài nếu cần.

Quy ước viết code:
1. Sử dụng camelCase cho tên biến và tên hàm.
2. Bổ sung thêm comment để giải thích code.
